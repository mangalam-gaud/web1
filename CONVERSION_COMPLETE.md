### **2. Setup Apache + PHP**

Option A - VirtualHost:
```apache
<VirtualHost *:80>
    ServerName smart.local
    DocumentRoot "/path/to/web1/client/public"
    <Directory "/path/to/web1/client/public">
        AllowOverride All
        Require all granted
    </Directory>
</VirtualHost>
```

Add to `hosts` file:
```
127.0.0.1 smart.local
```

### **3. Setup Database**
```bash
1. Open your MySQL admin tool (or MySQL CLI)
2. Create database: smart_influencing
3. Import php/database.sql
4. Update php/config.php with your DB credentials
```

### **4. Open Website**
```
http://smart.local/
or
http://localhost/web1/
```
